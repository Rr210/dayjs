## 仓库简介

* 该软件开发主要是用于前端知识的学习. 方便切换题目, 本地记录学习天数
* 前端题库来源`前端知识每日3+1`

## 开发技术

* 使用`python+tkinter`
* `exe`打包工具为`pyinstaller`
